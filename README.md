# 灵析个人量化投研平台

一个功能完整的个人量化投研平台，集因子挖掘、策略研究、投资组合分析于一体。

## 🚀 功能特色

### 因子研究
- **遗传规划算法**: 自动挖掘高质量因子
- **强化学习算法**: 智能优化因子组合
- **混合算法**: 结合多种算法的优势
- **实时日志**: 监控因子挖掘全过程
- **因子评估**: IC、ICIR、综合评分等多维度评估

### 策略研究
- **多种策略模板**: Top-K Drop、动量策略、均值回归、CTA策略
- **参数优化**: 灵活调整策略参数
- **回测系统**: 完整的回测框架
- **性能分析**: 夏普比率、最大回撤、胜率等指标
- **可视化图表**: 净值曲线、收益分布等

### 投资组合分析
- **资产配置**: 灵活的资产权重配置
- **风险分析**: 波动率、最大回撤、夏普比率
- **多样化分析**: 多样化比率、相关性分析
- **实时监控**: 组合净值实时跟踪
- **优化建议**: 基于风险收益的优化建议

### 数据分析中心
- **多维度分析**: 市场行情、因子表现、组合分析
- **可视化图表**: 折线图、柱状图、饼图、K线图
- **实时监控**: 关键指标实时监控
- **数据表格**: 详细数据展示
- **时间序列**: 支持不同时间范围分析

## 📖 功能使用指南

### 1. 因子研究功能

#### 使用步骤：
1. **进入因子研究页面**
   - 从首页点击"立即体验"
   - 在功能中心选择"因子研究"

2. **选择挖掘算法**
   - 遗传规划 (Genetic Programming)
   - 强化学习 (Reinforcement Learning)
   - 混合算法 (Hybrid)

3. **开始因子挖掘**
   - 点击"开始挖掘"按钮
   - 观察实时日志输出
   - 等待挖掘过程完成

4. **查看结果**
   - 查看发现的高质量因子
   - 分析IC、ICIR、综合评分等指标
   - 复制因子表达式使用

#### 核心指标说明：
- **IC (Information Coefficient)**: 信息系数，衡量因子预测能力
- **ICIR (IC Information Ratio)**: 信息比率，衡量因子稳定性
- **综合评分**: 综合考虑多个维度的评分
- **复杂度**: 因子表达式的复杂程度

### 2. 策略研究功能

#### 使用步骤：
1. **进入策略研究页面**
   - 在功能中心选择"策略研究"

2. **选择策略类型**
   - Top-K Drop策略
   - 动量策略
   - 均值回归策略
   - CTA策略

3. **配置策略参数**
   - 调整各策略的特定参数
   - 设置持有期、阈值等

4. **运行回测**
   - 点击"运行回测"按钮
   - 等待回测计算完成

5. **分析结果**
   - 查看总收益率、年化收益率
   - 分析夏普比率、最大回撤
   - 观察净值曲线走势

#### 策略说明：
- **Top-K Drop**: 选择表现最好的K只股票，去除表现最差的N只
- **动量策略**: 基于价格动量选择股票
- **均值回归**: 基于价格偏离均值的程度选择股票
- **CTA策略**: 商品交易策略，基于技术指标

### 3. 投资组合分析功能

#### 使用步骤：
1. **进入投资组合分析页面**
   - 在功能中心选择"投资组合分析"

2. **选择资产**
   - 从可用资产列表中选择
   - 调整各资产的权重配置

3. **分析组合**
   - 点击"分析组合"按钮
   - 等待组合分析计算

4. **查看分析结果**
   - 查看组合概览指标
   - 分析资产配置分布
   - 观察净值走势图表

#### 核心指标：
- **总收益率**: 投资组合的整体收益
- **年化收益率**: 年化后的收益率
- **夏普比率**: 风险调整后的收益指标
- **最大回撤**: 投资组合的最大亏损幅度
- **波动率**: 投资组合的收益波动程度
- **多样化比率**: 投资组合的多样化程度

### 4. 数据分析中心

#### 使用步骤：
1. **进入数据分析中心**
   - 在功能中心选择"数据分析中心"

2. **选择分析类型**
   - 市场行情
   - 因子表现
   - 组合分析

3. **设置时间范围**
   - 1天、1周、1个月、3个月、1年

4. **查看分析结果**
   - 关键指标概览
   - 多维度图表展示
   - 详细数据表格

## 🔧 自定义配置

### 修改配置文件
编辑 `config.py` 文件可以调整各种参数：

```python
# 数据配置
DATA_DIR = "data"
RESULTS_DIR = "results"

# 遗传规划配置
GP_POPULATION_SIZE = 100
GP_GENERATIONS = 50

# 强化学习配置
RL_EPISODES = 1000
RL_LEARNING_RATE = 0.001
```

class MyStrategy(BaseStrategy):
    def __init__(self):
        super().__init__()
        
    def generate_signals(self, data):
        # 实现策略逻辑
        pass
